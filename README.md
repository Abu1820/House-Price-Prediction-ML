# House Price Prediction

A simple Machine Learning project that predicts house prices using Linear Regression.

## Dataset Features

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

## Tools Used

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab

## Project Workflow

1. Data Preprocessing
2. Feature Scaling
3. Train-Test Split
4. Linear Regression Model Training
5. Model Evaluation
6. House Price Prediction

## Results

- R² Score: 0.649
- Successfully predicted house prices using Linear Regression

Input:
- Area: 5000 sq.ft
- Bedrooms: 3
- Bathrooms: 2
